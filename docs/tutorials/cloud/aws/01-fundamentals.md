# AWS Cloud Fundamentals

## Introduction
AWS is the leading cloud provider. This module introduces core services.

## Core Services

### EC2 (Elastic Compute Cloud)
Virtual servers in the cloud.
- **Instance Types**: T2/T3 (general), C5 (compute), etc.
- **AMI**: Amazon Machine Image.

### VPC (Virtual Private Cloud)
Your isolated network in the cloud.
- **Subnets**: Public (Internet Gateway) vs Private (NAT Gateway).
- **Security Groups**: Stateful firewalls for instances.

### S3 (Simple Storage Service)
Object storage.
- **Buckets**: Containers for objects.
- **Classes**: Standard, IA, Glacier.

### IAM (Identity and Access Management)
- **Users, Groups, Roles**.
- **Policies**: JSON documents defining permissions.
