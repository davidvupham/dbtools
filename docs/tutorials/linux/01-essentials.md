# Linux Essentials for DevOps

## Introduction
Linux forms the backbone of most DevOps infrastructure. This module covers the essential command-line skills you will use daily.

## Key Skills to Master

### File and Directory Management
- `ls`: List directory contents
- `cd`: Change directory
- `mkdir`: Make directory
- `rm`: Remove files/directories
- `cp`: Copy files
- `mv`: Move/rename files

### File Permissions and Ownership
Understanding `chmod`, `chown`, and `umask` is critical for security and access control.

### Process Management
- `ps`: View status of processes
- `top`/`htop`: Monitor system resources
- `kill`: Terminate processes

### System Monitoring
- `df`/`du`: Disk usage
- `free`: Memory usage
- `uptime`: System uptime

### Text Processing
- `grep`: Search text
- `awk`: Pattern scanning and processing
- `sed`: Stream editor
