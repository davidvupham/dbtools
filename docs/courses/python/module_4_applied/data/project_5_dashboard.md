# Project 5: COVID-19 Data Dashboard

**Goal**: Analyze and visualize real-world pandemic data.

## Learning Objectives

* Pandas DataFrames
* Data Cleaning
* DuckDB SQL Queries
* Visualization

## Requirements

1. **Ingest**: Load CSV data (e.g., JHU COVID dataset).
2. **Clean**: Handle missing values, convert dates.
3. **Analyze**:
    * Calculate daily new cases from cumulative data.
    * Find the top 5 countries by infection rate.
4. **Visualize**: Plot the trend line for a specific country.
