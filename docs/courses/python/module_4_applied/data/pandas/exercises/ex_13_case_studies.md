# Exercises — 13_case_studies: Case studies

## Goal
Combine multiple pandas skills into a realistic mini-analysis.

## Exercises

1. Generate `orders = make_orders(n=1000, seed=42)` and ensure `order_date` is datetime.

2. Deliverable A — Category report
   - Build a table with per-category:
     - `revenue_total`
     - `orders` (nunique order_id)
     - `items` (sum quantity)

3. Deliverable B — Monthly category pivot
   - Create a monthly revenue pivot (index: month start, columns: category).

4. Deliverable C — Top customers
   - Produce top 10 customers by total revenue.

## Deliverable
Paste the 3 tables (or `head()`/`tail()` if large).
