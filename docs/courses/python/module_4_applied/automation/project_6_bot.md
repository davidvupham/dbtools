# Project 6: Invoice Processing Bot

**Goal**: Automate the extraction of data from PDF invoices.

## Learning Objectives

* RPA (Robotic Process Automation)
* Robocorp
* PDF Parsing

## Requirements

1. **Trigger**: Watch a folder for new PDF files.
2. **Extract**: Open PDF, find "Invoice Number" and "Total Amount".
3. **Save**: Write extracted data to a CSV file or Excel sheet.
4. **Archive**: Move processed PDF to an "Archive" folder.
