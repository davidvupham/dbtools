# Control Flow

## Conditional Statements (`if`)

Used to make decisions.

```python
x = 10
if x > 0:
    print("Positive")
elif x < 0:
    print("Negative")
else:
    print("Zero")
```

## Loops

### `for` Loop

Iterate over a sequence.

```python
fruits = ["apple", "banana"]
for fruit in fruits:
    print(f"I like {fruit}")
```

### `while` Loop

Repeat while condition is true.

```python
count = 5
while count > 0:
    print(count)
    count -= 1
print("Blastoff!")
```
