## Overview

Ansible AWX is the open-source upstream project for Red Hat Ansible Automation Platform. It provides a web-based user interface, REST API, and task engine for Ansible. It allows you to manage Ansible playbooks, inventories, and credentials centrally, enabling enterprise-wide automation.

This documentation covers the architecture, best practices, and operations for maintaining Ansible AWX environments.
