# Release Readiness Checklist

- [ ] Backups verified and restorable
- [ ] Rollback procedure documented and tested
- [ ] Monitoring and alerts configured
- [ ] Security review completed and sign-offs recorded
- [ ] Runbook updated for new/changed procedures
- [ ] Load/performance testing completed (as applicable)
- [ ] Stakeholder communications prepared (plan + recipients)
- [ ] CAB approval (if applicable)
